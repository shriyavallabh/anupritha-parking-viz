Connect to Google Sheets Setup required

Paste the Google Apps Script below into your sheet, deploy it, and enter the URL here. Once connected, the form will write directly to the sheet and the table below will reflect any edits you make in the sheet within 1–2 minutes.

  1. Open your Google Sheet → Extensions → Apps Script.
  2. Delete any existing code. Paste the script below and click Save.
// Anupritha B — Vehicle Registry
// Handles form submissions from the website.
// Deploy as Web App: Execute as Me, Access: Anyone.

function doGet(e) {
  var action = (e && e.parameter && e.parameter.action) || '';
  var ss     = SpreadsheetApp.getActiveSpreadsheet();
  var sheet  = ss.getSheetByName('Vehicle Registry');

  if (action === 'write') {
    var today = Utilities.formatDate(
      new Date(), Session.getScriptTimeZone(), 'dd/MM/yyyy'
    );
    sheet.appendRow([
      e.parameter.flat        || '',
      e.parameter.owner       || '',
      '',                          // Tenant (admin fills if needed)
      'Owner',                     // Resident_Type
      e.parameter.vehicleType || '',
      e.parameter.makeModel   || '',
      e.parameter.regNumber   || '',
      'Active',
      today
    ]);
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'ok' }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  return ContentService
    .createTextOutput(JSON.stringify({ status: 'ready' }))
    .setMimeType(ContentService.MimeType.JSON);
}
  1. Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy → Authorise.
  2. Copy the Web App URL and paste it below.
Script URL saved. Form submissions will now write directly to your Google Sheet.
